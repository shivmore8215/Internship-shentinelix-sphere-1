## Quick Start

### Install in Chrome/Edge

1. **Open Extensions Page**
   ```
   Chrome: chrome://extensions/
   Edge: edge://extensions/
   ```

2. **Enable Developer Mode**
   - Toggle the switch in the top-right corner

3. **Load Extension**
   - Click "Load unpacked"
   - Navigate to and select the `NoHijacking` folder
   - Click "Select Folder"

4. **Done!**
   - You'll see the üõ°Ô∏è NoHijacking extension
   - A notification will confirm activation

## Note About Icons

The extension currently uses placeholder icons. For a production version, you would add:
- `icons/icon16.png` (16x16 pixels)
- `icons/icon48.png` (48x48 pixels)  
- `icons/icon128.png` (128x128 pixels)

You can create these using any image editor with a shield/clipboard design.

## Troubleshooting

**Extension won't load?**
- Make sure you selected the `NoHijacking` folder (not a parent folder)
- Check that all files are present

**Not detecting hijacking?**
- Make sure you're on a webpage (not chrome:// pages)
- Check browser console for errors
- Refresh the page after installing

**No notifications?**
- Check browser notification permissions
- Look for on-page alerts instead
